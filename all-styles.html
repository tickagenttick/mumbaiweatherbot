Language Indicator — Style Reference

Six minimalist treatments for the top-right language tag. Click Simulate Language Change to see each indicator blink for attention, then settle into its resting state for the duration of the video.

English

How the animation works: When a new video loads in a different language, the indicator briefly pulses (~1.6s: two quick flashes) to draw the eye, then returns to a calm resting state for the remainder of the video. The resting state is low-contrast and non-distracting — the flash is the only moment it commands attention.

Recommended spec: Animation duration 1.4–1.8s, two visible pulses, easing cubic-bezier(.4,0,.2,1) or a slight spring for playful variants. Resting opacity 80–95%, font-size 11–12px, uppercase tracking ~0.08em for code variants.
